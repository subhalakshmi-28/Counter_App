# Counter_App

A simple **Counter App** built using **HTML, CSS, and JavaScript**. This project demonstrates basic DOM manipulation and JavaScript functions.

## 🚀 Live Demo

**Demo Link:** https://subhalakshmi-28.github.io/Counter_App/

## ✨ Features

* ➕ Increase the counter by 1
* ➖ Decrease the counter by 1
* 🔄 Reset the counter to 0
* 📱 Simple and responsive user interface
* 🎨 Clean and beginner-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## 📖 How It Works

1. The counter starts at **0**.
2. Click **+1** to increase the counter.
3. Click **-1** to decrease the counter.
4. Click **Reset** to set the counter back to **0**.

## 🎯 Learning Objectives

This project helps beginners understand:

* HTML page structure
* CSS styling
* JavaScript variables and functions
* DOM manipulation
* Event handling using `onclick`
